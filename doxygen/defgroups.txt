
/* These groups will be listed in the same order in the documentation */

/**
 * @defgroup pub Definitions for user application code.
 *
 * These definitions can be used in user application code. They may act on
 * various types of arguments and produce machine code. Arguments are checked
 * and explicit error messages are emitted.
 */

/**
 * @defgroup dev Definitions for HWA developper code.
 *
 * These definitions can be used to add HWA support to a device.
 */

/**
 * @defgroup priv Definitions for HWA implementation code.
 *
 * These instruction are used internally by HWA. There is no argument checking
 * and no error processing.
 */
