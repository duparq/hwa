HWA
HardWare Abstractor
Configure a watchdog timer
hwa_config( hw_wdog0,
[timeout, 16ms
| 32ms
| 64ms
| 125ms
| 250ms
| 500ms
| 1s
| 2s
| 4s
| 8s,]
action, none
| irq
| reset
| irq_or_reset,
);

When action is irq_or_reset the watchdog IRQ is enabled. It is disabled by hardware after a timeout occurs. Then, if you do not re-enable the IRQ before a new timeout occurs, the device will be reset.

Turn watchdog on/off
hw/hwa_turn( hw_wdog0, on
| off );
Clear watchdog timer

This issues a wdr instruction:

hw_clear( hw_wdog0 );
Watchdog status

The only flag that is available is the irq flag.

hw_stat_t( hw_wdog0 ) st = hw_stat( hw_wdog0 );
if ( st.irq )
hw_toggle( hw_pin_pa0 );

A class _wdoga object is a watchdog timer
