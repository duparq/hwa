HWA
HardWare Abstractor
_c8a

A class _c8a object is an 8-bit counter/timer with two compare units and waveform generators.

Name of a compare unit

The compare units (class _ocua) are named compare0 and compare1, corresponding to OCRxA and OCRxB registers in Atmel terminology.

#define COUNTER hw_counter0
#define COMPARE hw_sub(COUNTER, compare0)
#if hw_id(COMPARE)==0
# error This counter has no compare unit!
#endif
Number of bits of the counting register
#if hw_bn( COUNTER ) != 16
# error You must choose a 16-bit counter!
#endif
Configure the counter
#define COUNTER hw_counter0
hwa_config( COUNTER,
clock, none
| syshz
| syshz_div_1
| syshz_div_8
| syshz_div_64
| syshz_div_256
| syshz_div_1024
| syshz_ext_rising
| syshz_ext_falling,
countmode, loop_up
| loop_updown,
[bottom, 0, ]
[top, fixed_0xFF
| max
| compare0,]
[overflow, at_bottom
| at_top
| at_max ]
);

top determines the maximum value the counter can reach before overflowing. If top is compare0, the counter counts from 0 to the value stored in the compare unit 0 (register OCRxA in Atmel terminology).

Optionnal parameter overflow tells when the overflow interrupt request will be thrown. If overflow is not stated, HWA will select an acceptable value according to the configuration of the comparison units. If overflow is stated, HWA will check its value.

Get the value of the counter
hw_read( COUNTER );
Set the value of the counter
hw/hwa_write( COUNTER, 42 );
Reset (set to 0) the value of the counter
hw/hwa_clear( COUNTER );

Note: this does not reset the prescaler.

Get the status of the counter

Available flags are:

  • overflow
  • compare0
  • compare1

These flags correspond to the interrupts requests the counter can trigger.

hw_stat_t(COUNTER) st ;
st = hw_stat( COUNTER );
if ( st.overflow ) {
hw_clear_irq( COUNTER, overflow );
n_overflows++ ;
}
