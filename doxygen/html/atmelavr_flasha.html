HWA
HardWare Abstractor
_flasha

A class _flasha object is an FLASH memory.

Read one byte
uint8_t byte = hw_read( FLASH, addr ); // Read byte at address addr
Store into dst count bytes read from Flash memory address addr
uint8_t dst[10];
uint8_t count = sizeof(dst);
hw_read_bytes( FLASH, dst, addr, count );
Clear page buffer
hw_command( FLASH, clear_page_buffer );
Erase page
intptr_t ptr = page_address ;
hw_command( FLASH, erase_page, ptr );
Write page buffer into Flash memory
intptr_t ptr = page_address ;
hw_command( FLASH, write_page, ptr );
