HWA
HardWare Abstractor
_io8a

A class _io8a object is an i/o port.

Configure the pin
#define PIN hw_pin_pa2
hw/hwa_config( PIN,
[mode, analog,
| digital,]
direction, input,
| output,
[pullup, on,
| off] );
Read a digital i/o pin
uint8_t p = hw_read( PIN );
Write a digital i/o pin
hw_write( PIN, 1 );
Toggle a digital i/o pin
hw_toggle( PIN );
