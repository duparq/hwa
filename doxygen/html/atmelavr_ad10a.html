_ad10a

A class _ad10a object is a 10-bit analog to digital converter.

Turn the ADC on / off
hw/hwa_turn( ADC, on | off );
Read the result of conversion
uint16_t adc = hw_read( ADC );
Start a conversion
hw/hwa_trigger( ADC );
Get the status of the converter

Available flags are:

  • irq
  • busy

The irq flag corresponds to the interrupt request the converter can trigger after a new data is available.

The busy flag indicates that a conversion is in progress.

hw_stat_t(ADC) st ;
st = hw_stat( ADC );
if ( st.irq ) {
hw_clear_irq( ADC );
adc = hw_read( ADC );
}
