HWA
HardWare Abstractor
_ocua

A class _ocua object is a counter/timer compare unit with waveform generator.

Name of the related counter
#define COMPARE hw_counter0_compare0
#define COUNTER hw_sup( COMPARE )
#if hw_bn(COUNTER) != 16
# error Only 16-bit counter is supported!
#endif
Name of the related i/o pin
#define COUNTER hw_counter0
#define COMPARE hw_sub( COUNTER, compare0 )
#define OUTPUT hw_io( COMPARE )
#if hw_id(OUTPUT) == hw_id(hw_pin_mosi)
# error OUTPUT pin is in conflict with pin MOSI
#endif
Configure the compare unit output
hw_config( COMPARE,
output, disconnected
| toggle_on_match
| clear_on_match
| set_on_match
| set_at_bottom_clear_on_match
| clear_at_bottom_set_on_match
| clear_on_match_up_set_on_match_down
| set_on_match_up_clear_on_match_down
);

Note: there is no checking against the counter configuration.

Configure the compare unit
hwa_config( COMPARE,
[update, at_bottom
| at_top
| at_max ]
[output, disconnected
| toggle_on_match
| clear_on_match
| set_on_match
| set_at_bottom_clear_on_match
| clear_at_bottom_set_on_match
| clear_on_match_up_set_on_match_down
| set_on_match_up_clear_on_match_down ]
);
Set the compare value
hw/hwa_write( COMPARE, 42 );
Get the compare value
uint16_t ocr = hw_read( COMPARE );
Trigger a compare event
hw/hwa_trigger( COMPARE );
