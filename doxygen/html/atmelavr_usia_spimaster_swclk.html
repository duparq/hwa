HWA
HardWare Abstractor
_usia_spimaster_swclk

A class _usia_spimaster_swclk object is an SPI master clocked by software.

Configure the SPI
#define SPI hw_spimaster0_swclk
hw_config( SPI );
Receive a byte from the SPI
#define SPI hw_spimaster0_swclk
uint8_t byte = hw_read( SPI );
Send a byte over the SPI
#define SPI hw_spimaster0_swclk
uint8_t byte = '\n';
hw_write( SPI, byte );
