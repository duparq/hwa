
Devices {#atmelavr_devices}
=======

There are the headers corresponding to the Atmel AVR devices HWA supports. These
files must be included with something like:

@code
#include <hwa/attiny24a_pu.h>
@endcode

 * @subpage attinyx4 :
   - `hwa/attiny24a_pu.h`
   - `hwa/attiny44a_pu.h`
   - `hwa/attiny84a_pu.h`

 * @subpage attinyx5 :
   - `hwa/attiny85_20pu.h`

 * @subpage atmegax8 :
   - `hwa/atmega328p_au.h`

 * @subpage atmegaxu4 :
   - `hwa/atmega32u4_mu.h`
