
HWA for Atmel AVR devices {#atmelavr}
=========================

There are the headers corresponding to the Atmel AVR devices HWA supports. These files must be included with something like:

@code
#include <atmel/avr/attiny24a_pu>
@endcode

 * @subpage attinyx4 :
   - attiny24a_pu
   - attiny44a_pu
   - attiny84a_pu

 * @subpage atmegax8 :
   - atmega328p_au
